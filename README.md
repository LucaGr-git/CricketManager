**CricketManager** is a graphical user interface (GUI) proof-of-concept (POC) application built using **Python** with the **Tkinter** library developed in 2021-2022. It is designed to help manage and analyse player stats for school cricket teams, making it easier for coaches, players, and support staff to track performance and team data.
